Histoire

Ce jeu est d’origine japonaise. Il se joue dans les collèges et lycées français depuis la fin du XIXème. En France il est couramment appelé "jeu du morpion. Le but du jeu : faire un alignement de cases soit horizontalement, soit verticalement, soit diagonalement en traçant un trait sur l’alignement réalisé (son trait mord les pions). Le plus souvent le jeu est pratiqué en utilisant une simple feuille de papier avec des carreaux. Jeu qui consiste pour chacun des deux adversaires à placer à tour de rôle un signe distinctif (croix ou cercle) sur du papier quadrillé pour s'efforcer d'obtenir le plus rapidement une file continue de cinq signes dans n'importe quelle direction

Règles

Pour jouer une partie de morpion, les règles sont simples: il suffit de tracer sur une feuille blanche une grille de 3 cases sur 3 (selon les variantes, il est possible d'augmenter le nombre de cases). Le but du jeu est d'aligner avant son adversaire 3 symboles identiques horizontalement, verticalement ou en diagonale.
